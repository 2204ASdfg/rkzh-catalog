# Размещение каталога на Tilda

## Рекомендуемая схема

Страница Tilda содержит встроенный каталог в iframe. Сам каталог размещается на отдельном статическом адресе, например `https://practices.example.ru/`.

Это сохраняет поиск, фильтры, детализацию кейсов и автоматическое обновление из исходной таблицы. Встраивание через T123 изолирует стили каталога от блоков Tilda.

## Что уже подготовлено

- Папка `site/` - готовый статический каталог.
- Папка `site/assets/` - четыре логотипа.
- `site/data.js` - резервная выгрузка базы; используется, если исходная таблица временно недоступна.
- `site/tilda-t123-embed.html` - код для блока T123.

Не загружайте на хостинг файл `start_site.cmd`; он нужен только для локального просмотра.

## Шаг 1. Опубликовать каталог

Подойдёт любой статический хостинг: GitHub Pages, Cloudflare Pages, Vercel, Netlify, Selectel Object Storage или хостинг вашего домена.

Загрузите содержимое папки `site/` в корень отдельного сайта или поддомена. После публикации проверьте, что открываются:

- `https://ВАШ_АДРЕС/`
- `https://ВАШ_АДРЕС/data.js`
- `https://ВАШ_АДРЕС/assets/presidential-academy.png`

Для корпоративного размещения рекомендован поддомен вида `practices.ваш-домен.ru`. Для него понадобится доступ к DNS домена, чтобы создать запись CNAME по инструкции выбранного хостинга.

## Шаг 2. Создать страницу на Tilda

1. В проекте Tilda создайте отдельную страницу, например «Библиотека лучших практик».
2. Уберите стандартные блоки страницы, либо оставьте только нужные шапку и подвал.
3. Добавьте блок «Другое» → `T123 HTML-код`.
4. Откройте файл `tilda-t123-embed.html`.
5. В строке `src="CATALOG_URL"` замените `CATALOG_URL` на фактический HTTPS-адрес каталога.
6. Вставьте весь код в T123 и опубликуйте страницу.
7. Откройте опубликованную страницу в обычном и мобильном браузере.

Код T123 исполняется на опубликованной странице, а не в редакторе Tilda.

## Шаг 3. Настроить адрес и меню

Рекомендуемый адрес страницы Tilda: `/practices` или `/library`.

Добавьте этот адрес в основное меню Tilda с названием «Библиотека практик». Если каталог размещён на собственном поддомене и должен открываться отдельно, используйте прямую ссылку на него в меню.

## Как обновлять практики

1. Внесите изменения в исходную Excel-таблицу на Яндекс Диске.
2. Сохраните файл, сохранив публичную ссылку на него неизменной.
3. На опубликованном каталоге нажмите «Обновить данные» либо обновите страницу браузера.
4. Проверьте изменённую практику через поиск.

При временной недоступности таблицы каталог использует встроенную резервную выгрузку, поэтому не становится пустым. После восстановления доступа повторное обновление страницы снова загрузит актуальную таблицу.

## Важные ограничения и проверка

- Публичная ссылка Яндекс Диска означает, что файл доступен по этой ссылке. Если данные должны быть закрытыми, потребуется отдельный серверный слой с авторизацией.
- Встроенный iframe подходит для каталога и внутреннего рабочего инструмента. Если важна индексация каждой практики поисковиками, каталог лучше открывать отдельным сайтом на поддомене, а не внутри Tilda.
- Сначала тестируйте после публикации Tilda: в режиме редактирования T123 не исполняет JavaScript.
- Если каталог не загрузился, проверьте HTTPS-адрес в `src` и доступность исходной таблицы по публичной ссылке.
